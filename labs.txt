Лабораторная 3
Написать программу, которая во введенной строке проверяет корректность раставленных скобок
двух видов: () []
abc  ( cde )  ( [ ( ] ) ) abc () 22
Алгоритм:
1. Создаём стек (std::stack)
2. Перебираем последовательно все символы в строке (std::string)
3. Если скобка открывающая ( или [ то добавляем на стек (push)
4. Если скобка закрывающая ) или ] проверяем что
	стек не пустой (!empty)
	извлекаем со стека последнюю скобку и проверяем что она нужно типа
5. В конце стек должен пуст (empty)

Лабораторная 4.1
Скачать, собрать и подключить Boost

Building boost using msvc
1. Download and extract boost lib
2. Make toolchain available in command line (using PATH) (skip for msvc)
3. .\bootstrap msvc or .\bootstrap
4. .\b2 --prefix=boost_msvc install

Building boost using gcc
1. Download and extract boost lib
2. Make toolchain available in command line (using PATH)
3. .\bootstrap gcc
4. .\b2 --prefix=boost_gcc install
